## Requirements

- **Python 3.11 or newer**
- An API key for at least one [supported provider](../../configuration/providers-and-models/)

## Install

<Tabs>
  <TabItem label="Installer">
    ```bash
    curl -fsSL https://kolega.dev/install-kolega-code.sh | sh
    ```
  </TabItem>
  <TabItem label="uv">
    ```bash
    uv tool install kolega-code
    ```
  </TabItem>
  <TabItem label="From source">
    ```bash
    git clone https://github.com/kolega-ai/kolega-code.git
    cd kolega-code
    uv sync --extra dev
    uv run kolega-code --version
    ```
  </TabItem>
</Tabs>

<Aside type="note">
The installer supports macOS and Linux. On Windows, use WSL for the terminal UI
or install directly with `uv tool install kolega-code`.
</Aside>

## Verify the install

Once installed, confirm the CLI is on your `PATH`:

```bash
kolega-code --version
```

Then check your project configuration:

```bash
kolega-code doctor --project .
```

`doctor` reports your project path, where session state is stored, whether Textual
is installed, your stored model selection, and whether the required API keys are
present. See [`doctor`](../../cli/doctor/) for details.

## Upgrade or uninstall

If you installed with the script or `uv tool install`, update the CLI with the
built-in command:

```bash
kolega-code update
```

Running the installer again also updates an existing install to the latest
released version. To uninstall, use uv:

```bash
uv tool uninstall kolega-code
```
