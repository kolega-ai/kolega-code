## What is Kolega Code?

**Kolega Code** is an AI coding agent you run in your terminal. It opens an
interactive [Textual](https://textual.textualize.io/) UI in your project, reads
and edits your files, runs commands, browses the web, and works through tasks the
way a pair-programmer would — all on your local machine, against the LLM provider
of your choice.

It also has a scriptable side: a one-shot `ask` command, JSON output for piping
into other tools, and resumable sessions.

<CardGrid stagger>
  <Card title="Interactive TUI" icon="laptop">
    A full terminal interface with a streaming conversation, tool output, logs,
    a built-in terminal, and a settings panel. Mention files with `@`, run
    commands with `/`.
  </Card>
  <Card title="Plan, then build" icon="pencil">
    Switch to **Plan mode** for a read-only planning pass, review the proposed
    plan, then drop into **Build mode** to implement it. Press `Shift+Tab` to
    toggle.
  </Card>
  <Card title="Bring your own model" icon="setting">
    Anthropic, OpenAI, Google, Moonshot, DeepSeek, and more. Assign different
    models to different roles — a long-context model for coding, a fast model
    for quick utility calls.
  </Card>
  <Card title="Scriptable" icon="seti:powershell">
    Run a single prompt with `kolega-code ask "…"`, emit structured `--json`
    events, and save or resume sessions for repeatable workflows.
  </Card>
  <Card title="Skills" icon="puzzle">
    Drop reusable workflows into `.agents/skills/` and invoke them as
    `/skill-name`. The agent loads the skill's instructions and bundled
    resources on demand.
  </Card>
  <Card title="Local-first" icon="approve-check">
    Runs against your real filesystem and terminal. Sessions and settings are
    stored locally, with API keys written using restrictive file permissions.
  </Card>
</CardGrid>
