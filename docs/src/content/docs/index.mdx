## What is Kolega Code?

**Kolega Code** is an AI coding agent that runs in your terminal. Point it at a
project directory and use it to inspect code, edit files, run commands, plan
changes, and work through development tasks on your local machine.

## Install

```bash
curl -fsSL https://kolega.dev/install-kolega-code | sh
```

## Direct install

```bash
uv tool install kolega-code
```

## Verify

```bash
kolega-code --version
```

## Start

```bash
kolega-code .
```

See [Installation](/kolega-code/getting-started/installation/) for requirements,
upgrade commands, and uninstall commands. See the
[Quick Start](/kolega-code/getting-started/quick-start/) for the first run.
